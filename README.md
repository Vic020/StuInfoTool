# StuInfoTool
A tool of DLNU Web, Using Python to fetch useful information,showing with desktop GUI.More quickly get info than using normal website.

## Functions
>
> Display term or whole grade 
>

>
> Output .xls
>
> Support SSL proxy
>

>
